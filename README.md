# Learning Java, 7th Ed.
Examples and exercises from the upcoming next edition of O'Reilly Media's Learning Java by Marc Loy, Patrick Niemeyer, and Daniel Leuck.

We're just getting our ducks in a row at the moment, but we'll be posting material for new chapters as they become available for early access readers.

If you're looking for the examples from previous editions, that content should still be available for both the [5th](https://github.com/l0y/learnjava5e) and the [6th](https://github.com/l0y/learnjava6e). (The [web](https://github.com/l0y/learnjava5e-web) examples from the 5th edition are also available.)

More coming soon!
